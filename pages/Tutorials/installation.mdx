# `benchkit` Installation

Requirements:
- [Bench AI Account ↗](https://bench-ai.com/signup)
- [python](https://www.python.org/downloads/) >= 3.10

## Installing from PyPI

import { Callout } from 'nextra/components'

<b>benchkit</b> can be installed via PyPI

<Callout>
    Currently only [PyTorch ↗](https://pytorch.org/get-started/locally/) >= 2.0.0 is supported
</Callout>

<Callout>
    We support macOS, Windows (including WSL), and Linux
</Callout>

```bash filename="> Terminal" copy
pip install benchkit
```

## Build from Source
clone your own personal copy of `benchkit`

```bash filename="> Terminal" copy
git clone https://github.com/bench-ai/benchkit.git
```
make a virtual environment
```bash filename="> Terminal" copy
python3.10 -m venv venv
source venv/bin/activate
```
install poetry(the build tool)
```bash filename="> Terminal" copy
pip install poetry
```
Install all requirements
```bash filename="> Terminal" copy
poetry install
```

## Verify the installation
```bash filename="> Terminal" copy
benchkit --version
```
