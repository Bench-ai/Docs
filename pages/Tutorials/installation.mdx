# Bench Kit Installation

Requirements:
- [Bench AI Account ↗](https://bench-ai.com/signup)
- [Python](https://www.python.org/downloads/) >= 3.10

## Installing from PyPI

import { Callout } from 'nextra/components'

<b>bench-kit</b> can be installed via PyPI

<Callout>
    Currently only [PyTorch ↗](https://pytorch.org/get-started/locally/) >= 2.0.0 is supported
</Callout>

<Callout>
    We support macOS, Windows (including WSL), and Linux
</Callout>

```bash filename="> Terminal" copy
pip install git+https://github.com/Bench-ai/BenchKit.git
```

Verify the installation by running:
```bash filename="> Terminal" copy
bench-kit --version
```
